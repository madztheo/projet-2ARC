Lancer le jeu sur un emulateur NES

  1. Téléchargez l'émulateur FCEUX
  2. Lancez l'executable fceux.exe
  3. Parcourez vos fichier: File > Open ROM...

  4. Ouvrez game.nes

Le jeu

Le but du jeu

Il faut détruite chacune des briques sans faire tomber la boule. Chaque brique nécessite d'être touchée une première fois avant de pouvoir être totalement détruite.

Si vous faites tomber la boule, vous recommencerez depuis le début.

Le jeu dispose actuellement de 2 niveaux.

Le score

Brique touchée +50
Brique détruite +100

Les touches

# Action Touche PC Touche NES
1 Pause Enter Start
2 Valider F A
3 Déplacer le palet vers la gauche ← ←
4 Déplacer le palet vers la droite → →

Les écrans

L'écran d'acceuil

S'affiche au lancement du jeu.

L'écran de transition entre les niveaux

S'affiche entre chaque niveau.

L'écran en cas de victoire

S'affiche si tous les niveaux ont été complété.

L'écran en cas d'echec

S'affiche si la boule est tombée.
